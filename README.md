### Forensic

###### Bengineer forensic
![image](https://github.com/Craglitch/WHCCTF-POC_Writeup/blob/main/IMG_20220103_105209.jpg?raw=true)

0. Download .7z
1. Buka termux ( saya guna termux )
2. Unarchive guna
```bash
7z x file.7z
```
3. Dapat file pcap
4. Guna termshark
```bash
termshark
```
5. Pilih file pcap
6. Mula analisa tcp stream
7. Dapat flags di row 4 string "what worry me ?".
